# JavaScript Loop Programs

This repository contains beginner-friendly JavaScript programs based on loops and number operations. 

## Programs Included

1. Print Numbers from 1 to N
2. Print Numbers from N to 1
3. Sum of N Numbers
4. Factorial of a Number
5. Fibonacci Series
6. Prime Number Check
7. Prime Numbers Between a Range
8. Armstrong Number
9. Sum of Digits
10. Count Digits

## Concepts Covered

* Variables
* Data Types
* User Input
* Conditional Statements
* Loops (`for`, `while`)
* Mathematical Operations
* Functions
* Number Manipulation

## Technologies Used

* HTML
* JavaScript
